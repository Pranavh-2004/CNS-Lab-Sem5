Challenge:
A global ISP’s “TrueEdge” core switch was left exposed during a maintenance window. Attackers believe the device hides an internal admin flag. You’ve been given a firmware update image and a traffic capture from when engineers “tested it.” Recover the flag hidden in the switch’s management plane.

Download the sasquatch file into the same directory as your trueedge_update.bin file and run PATH="$PWD:$PATH" before you start using binwalk to find out whats in the firmware update file

View Hint: Free Hint
Decrypt the pcap file. check the files you got from binwalk to find a key
View Hint: Free hint
The flag is isfcr{X-Auth}

if you know what x-auth is, then you know the answer

Solution:
isfcr{specterAuth2025}
